# 🚀 Overview

CRM Pro is a recruiter-level full-stack SaaS CRM application designed with a premium modern UI and real-world business workflow features.

The platform helps businesses:

* Manage incoming leads
* Track lead conversion
* Organize workflow pipelines
* Handle client communication
* Visualize analytics
* Automate tasks and emails

Built using React.js, Node.js, Express.js, and MongoDB Atlas.

---

# 💡 Problem Statement

Many startups, freelancers, and small businesses struggle with organizing leads, follow-ups, and client communication efficiently.

Without a CRM system, businesses often face:

* Lost leads
* Poor workflow management
* Lack of analytics
* Inefficient follow-ups
* Scattered client information

Modern businesses require centralized systems to manage clients and improve productivity.

---

# 🎯 Solution

CRM Pro solves these challenges by providing:

* Lead management dashboard
* Analytics & KPI tracking
* Drag-and-drop workflow pipeline
* Calendar & task management
* Email automation
* File upload management
* Premium responsive SaaS interface

The project was designed to simulate how real-world CRM platforms operate.

# ⚙️ Deployment Configuration

## Vercel Settings

```bash

Framework Preset: Create React App

Build Command:
npm run build

Output Directory:
build

Install Command:
npm install
```

### Environment Variable

```env
CI=false
```

---

## Render Settings

```bash
Runtime: Node

Build Command:
npm install

Start Command:
node server.js
```

### Environment Variables

```env
MONGO_URI=your_mongodb_uri

JWT_SECRET=your_secret_key

EMAIL_USER=your_email

EMAIL_PASS=your_password
```

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/srishtii26/FUTURE_FS_02.git
```

---

## 2️⃣ Navigate Into Project

```bash
cd FUTURE_FS_02
```

---

## 3️⃣ Install Frontend Dependencies

```bash
cd client
npm install
```

---

## 4️⃣ Install Backend Dependencies

```bash
cd ../server
npm install
```

---

## 5️⃣ Create Environment Variables

Create a `.env` file inside the server folder:

```env
MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret_key

EMAIL_USER=your_email

EMAIL_PASS=your_password
```

---

## 6️⃣ Run Backend

```bash
npm start
```

---

## 7️⃣ Run Frontend

```bash
cd client
npm start
```
